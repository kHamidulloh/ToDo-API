# ToDo API

A simple RESTful API for managing tasks (ToDo), built with Go using the Gin web framework, GORM for ORM, and PostgreSQL as the database.

---

## Tech Stack

- Go
- Gin (Web Framework)
- GORM (ORM for Go)
- PostgreSQL
- godotenv (for environment variable management)
